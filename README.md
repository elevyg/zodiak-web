# Zodiak Web

Sitio one-page en Next.js para Zodiak Climb.

## Requisitos

- Node.js 20+
- npm 10+

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Contenido editable (sin CMS)

- `content/site.es.json`: metadata, navegación, hero, marca, contacto, redes.
- `content/products.es.json`: productos y características.
- `content/faq.es.json`: preguntas frecuentes.
- `content/gallery.json`: imágenes de galería.
- `content/about.md`: narrativa de marca.
- `public/images/*`: assets visuales.

## Nota de despliegue

El proyecto está listo para desplegar en Vercel como app Next.js estándar.
